# 咖喱小助手 - 写作资源管理库

网文爆款素材管理工具，帮助作者管理、搜索、获取灵感。

## 功能特点

- 🎯 **每日灵感** - 随机推荐文章 + 创意组合建议
- 📂 **分类浏览** - 57个分类，快速定位
- 🔍 **多维搜索** - 关键词/分类/爆点筛选
- 📊 **数据统计** - 分类饼图 + 标签热度
- 📥 **导入爆文** - 支持TXT/ZIP/粘贴导入

访问：[Streamlit Cloud 链接]

```bash
# 安装依赖
pip install -r requirements.txt

# 运行应用
streamlit run app.py
```
